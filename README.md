# QA Learning Lab

Repository created to practice concepts of Software Quality Assurance and Web Test Automation using Cypress.

## Objectives

* Learn automated testing fundamentals
* Practice web application validation
* Implement good testing practices
* Build a QA portfolio

## Stack

* Cypress
* JavaScript
* Node.js

## Test Scenarios

### Login

* Valid login
* Invalid login

### Cart

* Add item
* Remove item
